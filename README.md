# `nogo`
## Minimalistic `go`-based `cli` app to interact with Notion

### Usage

```shell
# list all commands
nogo h
```

#### `nogo` to-do functionality
```shell
# show the to-do list
nogo ls todo

# add a new to-do item
nogo todo add "Change the world"
```
